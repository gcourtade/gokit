usage: nucproSBM.py [-h] [--CA_rad CA_RAD] [--CAcom] [--CB_rad CB_RAD]
                    [--Kb KB] [--Ka KA] [--Kd KD] [--cutoff CUTOFF]
                    [--scaling SCALING] [--attype ATTYPE] [--CBcom] [--CBfar]
                    [--dsb] [--native_ca NATIVE_CA] [--grotop GROTOP]
                    [--aa_pdb AA_PDB] [--pdbgro PDBGRO] [--w_native W_NATIVE]
                    [--pl_map] [--skip_glycine] [--btmap] [--mjmap]
                    [--CB_radii] [--P_rad P_RAD] [--S_rad S_RAD]
                    [--Bpu_rad BPU_RAD] [--Bpy_rad BPY_RAD] [--nKb NKB]
                    [--nKa NKA] [--nKd NKD] [--P_nKd P_NKD]
                    [--Bpu_pos BPU_POS] [--Bpy_pos BPY_POS] [--S_pos S_POS]
                    [--P_pos P_POS] [--all_chains] [--pistacklen PISTACKLEN]
                    [--debye] [--T T] [--CBcharge] [--Kr KR] [--iconc ICONC]
                    [--irad IRAD] [--dielec DIELEC] [--hpstrength HPSTRENGTH]
                    [--ext_conmap EXT_CONMAP] [--interaction] [--dswap]
                    [--hphobic] [--hpdist HPDIST] [--interface]
                    [--custom_nuc CUSTOM_NUC]

Generate GROMACS and OPTIM potential files for enhanced SBM models.

optional arguments:
  -h, --help            show this help message and exit
  --CA_rad CA_RAD, -CA_rad CA_RAD
                        Radius for C-alpha atom. Default=4.0
  --CAcom, -CAcom       Place C-alpha at COM of backbone
  --CB_rad CB_RAD, -CB_rad CB_RAD
                        Statistically derived values by default for attype 2.
  --Kb KB, -Kb KB       Kbond
  --Ka KA, -Ka KA       Kangle
  --Kd KD, -Kd KD       Kdihedral
  --cutoff CUTOFF, -cutoff CUTOFF
                        Cut-off for contact-map generation
  --scaling SCALING, -scaling SCALING
                        Scaling for mapping to all-atom contact-map.
  --attype ATTYPE, -attype ATTYPE
                        Number of atom types. E.g. 1 for CA, 2 for CA and CB
  --CBcom, -CBcom       Put CB at center of mass of side-chain (no hydrogens)
  --CBfar, -CBfar       Place C-beta on farthest non-hydrogen atom.
  --dsb, -dsb           Use desolvation barrier potential for contacts.
  --native_ca NATIVE_CA, -native_ca NATIVE_CA
                        Native file with only C-alphas. Just grep pdb.
  --grotop GROTOP, -grotop GROTOP
                        Gromacs topology file output name.
  --aa_pdb AA_PDB, -aa_pdb AA_PDB
                        all-atom pdbfile e.g. 1qys.pdb
  --pdbgro PDBGRO, -pdbgro PDBGRO
                        Name for .gro file.
  --w_native W_NATIVE, -w_native W_NATIVE
                        Write native files, CA-CB_P-S-B from all atom PDB
                        file.
  --pl_map, -pl_map     Plot contact map for two bead model
  --skip_glycine, -skip_glycine
                        Skip putting Cbeta on glycine
  --btmap, -btmap       Use Betancourt-Thirumalai interaction matrix.
  --mjmap, -mjmap       Use Miyazawa-Jernighan interaction matrix.
  --CB_radii, -CB_radii
                        External contact map in format chain res chain res
  --P_rad P_RAD         Radius for Backbone Phosphate group bead. Default=3.7A
  --S_rad S_RAD         Radius for Backbone Sugar group bead. Default=3.7A
  --Bpu_rad BPU_RAD     Radius for N-Base Purine bead. Default=1.5A
  --Bpy_rad BPY_RAD     Radius for N-Base Pyrimidine bead. Default=1.5A
  --nKb NKB             Kbond for RNA/DNA
  --nKa NKA             Kangle for RNA/DNA. Default=20
  --nKd NKD             Kdihedral for Bi-Si-Si+1-Bi+1. Default=0.5
  --P_nKd P_NKD         Kdihedral for Backbone Pi-Pi+1-Pi+2-Pi+3. Default=0.7
  --Bpu_pos BPU_POS     Put input atom of Purine
                        [N1,C2,H2-N2,N3,C4,C5,C6,O6-N6,N7,C8,N9,COM] as
                        position of B. Default=COM(Center_of_Mass)
  --Bpy_pos BPY_POS     Put input atom of Pyrimidine
                        [N1,C2,O2,N3,C4,O4-N4,C5,C6,COM] as position of B.
                        Default=COM(Center_of_Mass)
  --S_pos S_POS         Put input atom of Sugar
                        [C1',C2',C3',C4',C5',H2'-O2',O3',O4',O5',COM] as
                        position of S . Default=COM(Center_of_Mass)
  --P_pos P_POS         Put input atom of Phosphate [P,OP1,OP2,O5',COM] group
                        as position of P. Default=COM(Center_of_Mass)
  --all_chains          Will Not remove identical chains.
  --pistacklen PISTACKLEN
                        pi-pi stacking length. Default=3.6A
  --debye               Use debye electrostatic term
  --T T                 System temperature. Default = 100K
  --CBcharge, -CBcharge
                        Put charges on CB for K,L,H,D,E
  --Kr KR               Krepulsion. Default=5.7A
  --iconc ICONC         Solvant ion conc. Default=0.1M
  --irad IRAD           Solvant ion rad. Default=1.4A
  --dielec DIELEC       Dielectric constant of solvant. Default=70
  --hpstrength HPSTRENGTH, -hpstrength HPSTRENGTH
                        Strength with which hydrophobic contacts interact.
  --ext_conmap EXT_CONMAP, -ext_conmap EXT_CONMAP
                        External contact map in format chain res chain res
  --interaction, -interaction
                        User defined interactions in file interaction.dat.
  --dswap, -dswap       For domain swapping runs. Symmetrised SBM is
                        generated.
  --hphobic, -hphobic   Generate hydrophobic contacts.
  --hpdist HPDIST, -hpdist HPDIST
                        Equilibrium distance for hydrophobic contacts.
  --interface, -interface
                        Takes input for Nucleiotide_Protein interface from
                        file nucpro_interface.input.
  --custom_nuc CUSTOM_NUC
                        Use custom non native DNA/RNA structure1.
